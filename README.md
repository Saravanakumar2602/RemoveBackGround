## ✨ Features

- **100% Client-Side Processing:** Images are processed directly in your browser. No data ever leaves your device, ensuring complete privacy.
- **Lightning Fast AI:** Uses highly optimized WASM and WebGL models to sever backgrounds within seconds.
- **Modern & Interactive UI:** Built with Framer Motion for buttery-smooth micro-interactions.
- **Drag & Drop Ready:** Intuitive file uploads with visual feedback.
- **HD Download Output:** Extract perfectly isolated subjects as high-resolution transparent PNGs.

## 💻 Tech Stack

This project is built using modern, cutting-edge web technologies:
* **Framework:** [Next.js 16](https://nextjs.org/) (React 19)
* **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
* **Animations:** [Framer Motion](https://www.framer.com/motion/)
* **AI Engine:** [@imgly/background-removal](https://img.ly/docs/background-removal/)
* **Icons:** [Lucide React](https://lucide.dev/)

## 🛠️ Local Development Setup

If you'd like to run the project locally or contribute, follow these easy steps:

### Prerequisites
Make sure you have Node.js (v18+) and npm installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Saravanakumar2602/RemoveBackGround.git
   cd RemoveBackGround/remove-bg-next
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```
